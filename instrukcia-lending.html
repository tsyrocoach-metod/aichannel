Мини-лендинг за вечер:
Claude + GitHub или Tilda

Как попросить Claude сверстать страницу вашей программы и разместить её по ссылке бесплатно. Без дизайнера, без кода.

01

Напишите промпт в Claude

Откройте claude.ai и опишите вашу программу. Чем конкретнее — тем точнее результат:

промпт для claude
Сверстай мини-лендинг для моей программы в виде одного HTML-файла. Программа: [название] Для кого: [опишите клиента — кто он, в какой ситуации] Результат: [что получит участник] Формат: [как проходит работа] Цена: [сумма] CTA: написать в Telegram @ваш_ник слово [кодовое слово] Стиль: лёгкий, минималистичный, светлый фон. Всё — в одном HTML-файле, без внешних зависимостей кроме Google Fonts.
Важно: просите Claude сделать всё в одном файле. Так его легко скачать и разместить — без лишних папок и CSS-файлов.
02

Скачайте и проверьте

1

Скачайте файл

Под ответом Claude найдите кнопку скачивания. Сохраните как index.html.

2

Откройте в браузере

Перетащите файл в Chrome или Safari. Проверьте: всё ли правильно, работают ли ссылки.

3

Что-то не так — скажите Claude

В том же чате: «Поменяй цвет», «Добавь блок с модулями», «Сделай текст короче». Получите новый файл.

03

Выберите, где разместить

GitHub Pages бесплатно
  • Полностью бесплатно
  • Ссылка вида имя.github.io/страница
  • Несколько страниц в одном месте
  • Идеально для конспектов и заданий
Tilda проще визуально
  • Бесплатный тариф — 1 сайт
  • Вставить HTML как блок
  • Свой домен на платном тарифе
  • Удобно если уже пользуетесь
04a

Размещение на GitHub Pages

1

Создайте аккаунт на github.com

Регистрация — 2 минуты.

2

Новый репозиторий

Нажмите «+» → «New repository». Название например landing. Обязательно выберите Public.

3

Загрузите файл

«Add file» → «Upload files» → загрузите index.html → «Commit changes».

4

Включите Pages

Settings → Pages → Branch: main → Save. Через 1–2 минуты появится ссылка.

Лайфхак: загружайте в тот же репозиторий разные файлы — modul1.html, zadanie2.html. Каждый получит свою ссылку. Идеально для материалов программы вместо Google Docs.
04b

Размещение на Tilda

1

Создайте проект на tilda.cc

Бесплатный аккаунт — один сайт.

2

Добавьте HTML-блок

В редакторе → «+» → блок «HTML» (категория Other) → вставьте содержимое файла.

3

Опубликуйте

Нажмите «Опубликовать» — страница получит адрес вида имя.tilda.ws.

+

Полезные запросы для доработки

Поменять дизайн

промпт
Поменяй цветовую схему: светлый фон, терракотовый акцент. Шрифт — более классический, с засечками.

Добавить блок

промпт
Добавь секцию с модулями программы после блока «для кого». Модули: [список]. Оформи в виде карточек.

Страница-конспект для участников

промпт
Создай красивую HTML-страницу-конспект. Тема: [тема]. Содержание: [текст или тезисы]. Стиль: лёгкий, читабельный, светлый фон. Один HTML-файл.
